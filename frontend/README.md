# 校园智航 · 动态导航前端

校园动态导航系统的 Vue 前端，目录固定为 `project/frontend`。

采用**深色科技风 + 玻璃拟态**视觉体系，按可展示、可汇报的产品原型设计，而非演示占位页。

## 界面构成

| 区域 | 组件 | 说明 |
|---|---|---|
| 顶部导航栏 | `TopNav.vue` | 品牌、当前场景、后端连接状态 |
| 行程规划 | `ControlPanel.vue` | 起终点选择、出行方式、时段 |
| 地图与路线 | `MapView.vue` | 地图/路线图层切换、缩放拖拽、HUD、图例 |
| 导航结果 | `ResultPanel.vue` | 距离、耗时、路径节点时间线、文字导航 |
| AI 辅助导航 | `AiAssistant.vue` | 聊天气泡、流式生成、重新生成 |

## 功能说明

- 起终点：可在全部校园地点中**自由选择**（支持搜索、一键对调）
- 出行方式：`步行` / `自行车`
- 时段模式：`普通时段` / `早高峰` / `午高峰` / `晚高峰`（与后端时段一致）
- 调整任一选项即**自动重新规划**，距离、耗时、路径实时联动
- 路线图层：仅示例场景 `桃蹊饭堂(C2) → 格致楼(S1)` 的普通/早高峰绘有红色路线，其余场景显示不画路线的普通校园地图；导航数据（距离/耗时/节点）始终由后端真实计算

### AI 辅助导航

- 将后端返回的路径节点自动转换为**自然语言导航说明**
- 一键生成 / 重新生成
- **打字机流式输出**效果，聊天气泡形式展示
- 若模型返回思考过程，可展开查看
- 数据来源：后端 `POST /api/chat`（DeepSeek）

## 接口对接

通过 Vite 代理（见 `vite.config.js`）转发到后端，无需处理跨域：

- `POST /api/navigation` —— 最短路径导航
- `GET /api/places` —— 全部校园地点（用于起终点选择）
- `POST /api/chat` —— AI 自然语言导航说明
- `GET /health` —— 后端健康检查

## 运行方式

先启动后端：

```bash
python main.py
```

再进入前端目录安装依赖并启动：

```bash
cd frontend
npm install
npm run dev
```

默认访问地址：`http://127.0.0.1:5173`
